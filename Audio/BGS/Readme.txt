The sounds in this folder only work if you have the FmodEX extention 
installed. A link to the extention can be found below.

https://www.pokecommunity.com/showthread.php?t=349496
Remember to credit Nuru Yuri if you use the FModEX extention

Remember to input the name of the file (e.g. "HeavyStorm.ogg")
into the value of the appropriate weather in the script. If
you change the name of one, the other will have to be changed
as well.

The audio files listed here go into the Audio/BGS folder.
